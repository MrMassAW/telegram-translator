lightbulb How to create a translation pair expand_more

1. Source — Add a channel or group where posts come from. The bot must already be added there. Use the + flow to verify and save the source.

2. Destinations — Open your source, then add one or more destination groups/channels and choose a target language (or “No translation” to forward without translating).

3. Options — Translate text in images (OCR + image render; experimental). Translate poll posts a new translated poll (votes are separate). Never translate these words keeps brands and names unchanged. Toggle rules per destination; use the dashboard switches to pause a whole source or a single pair.

Costs (default) — 1¢ per translated text message; 10¢ per image when image translation is on and that image is processed. Buy and track balance under Account (header shows your balance).
